## アプリ開発指示
住宅ローンの借入額から月々の支払いをシミュレートし、物件を比較するWebアプリを作成する。

## 参考資料: 計算ロジック
計算ロジック解説

---
計算ステップ（calculateLoan の流れ）

1. 資金構成（calcFundComposition）

資金構成 = 物件価格 + 諸経費

ローン総額の基準となる値です。

---
2. 通常月返済額（calcMonthlyPayment）— 元利均等返済

r = 金利(%) / 100 / 12        ← 月利
n = ローン期間(年) × 12       ← 返済回数
元金 = 資金構成 − ボーナス元金

月返済額 = 元金 × r / (1 − (1+r)^(−n))

これは元利均等返済（毎月の支払額が一定）の標準公式です。

金利が 0% の場合は上式が 0/0 になるため、単純に 元金 / n で計算しています（loanCalc.ts:80）。

---
3. ボーナス払い（calcBonusPayment）— 半年ごと元利均等返済

r6 = 金利(%) / 100 × 6/12     ← 半年利（6ヶ月分）
m  = ローン期間(年) × 2       ← ボーナス返済回数
ボーナス1回 = ボーナス元金 × r6 / (1 − (1+r6)^(−m))

通常の月返済とは別の元金プールに対して、同じ元利均等公式を半年単位で適用します。ボーナス元金が 0 の場合は 0 を返します。

---
4. 月々支払合計（calcMonthlyTotal）

月々合計 = 通常月返済額 + 管理費 + 修繕積立費 + その他費用

ローン返済以外の毎月コストを合算した「実質的な毎月の出費」です。

---
null 安全設計

- 計算に必要な値が未入力（undefined）や非有限値（NaN、Infinity）の場合、各関数は null を返します（loanCalc.ts:40-42）
- 表示時は formatYen() / formatNumber() が null を '—' に変換します（loanCalc.ts:170-181）
